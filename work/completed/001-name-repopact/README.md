Adopt RepoPact as the public identity, establish the canonical local repository
at `C:\Projects\repopact`, and publish the initial repository to GitHub.

## Why RepoPact

The name describes the central idea directly: the repository is the durable pact
between humans, agents, plans, evidence, and architecture. It is pronounceable,
searchable, and specific to repository-native coordination.

## Collision review

- No exact GitHub repository or account collision was found during review.
- No exact package was found on PyPI, npm, crates.io, or NuGet.
- No DNS records were found for the primary `.com`, `.dev`, `.org`, `.io`, or
  `.ai` names checked.
- A recently published adjacent project named `Pact` overlaps conceptually but
  does not use the RepoPact identity.

This is a practical naming review, not formal trademark clearance.
